# Documentation

## Installing
```bash
git clone https://github.com/femioladipo/Fython

cd Fython

python2 interpreter.py examples/example1.fy # runs example1
```
## Available Commands
- INT [X]: mandatory argument
- ADD
- SUB
- SWAP
- DUP
- POP
- CALL
- RET
- JGE
- JEQ
- PRINT
- EXIT
